This is a simple script to change your status/presence under your name in Discord to whatever is playing in an MPD instance.

### Node Library Requirements
- discord.io
- mpd

### Format tags
|Tag       |Description |
|----------|------------|
|`%title%` |Song title  |
|`%artist%`|Song artist |
|`%album%` |Album       |
|`%genre%` |Genre       |
